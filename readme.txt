== Description ==
WPCO stores your option values in the database.
Import/Export your options to json.
Install and use the shortcode easily.

== Installation ==
1. Download and extract plugin files to a wp-content/plugin directory.
2. Activate the plugin through the WordPress admin interface.
3. Done !
